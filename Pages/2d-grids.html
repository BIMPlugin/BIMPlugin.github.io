What is 2D Grids?

2D Grids is a productivity tool that converts visible 3D grid extents to 2D grid extents in the current Revit view.

Instead of manually changing each grid from 3D to 2D, the tool updates all grids in the active plan or view with a single click.

Designed for everyday BIM workflows, 2D Grids helps clean up documentation views, improves consistency, and saves time when preparing drawings.

How to Use 2D Grids?

  1. Open a plan or view that contains grids with 3D extents.
  2. Click the 2D Grids button in the BIM Plugin ribbon.
  3. BIM Plugin will automatically convert all visible grids in the current view to 2D extents.

Note: The tool affects grids visible in the active view.
