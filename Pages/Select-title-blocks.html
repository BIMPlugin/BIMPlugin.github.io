What is Select Title Blocks?

Select Title Blocks is a productivity tool that automatically selects the title blocks placed on one or more selected sheets in Autodesk Revit.

Instead of opening each sheet and manually locating its title block, the tool instantly selects the corresponding title blocks, making it easier to edit, inspect, or modify them.

Designed for everyday BIM workflows, Select Title Blocks saves time, reduces repetitive work, and simplifies title block management across multiple sheets.

How to Use Select Title Blocks?

  1. Select one or more sheets in the Revit Project Browser.
  2. Click the Select Title Blocks button in the BIM Plugin ribbon.
  3. BIM Plugin will automatically select the title blocks that belong to the selected sheets.

Note: Only title blocks placed on the selected sheets will be selected.
