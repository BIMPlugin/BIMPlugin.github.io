What is 3D Section Box?

3D Section Box is a productivity tool that creates a section box around selected elements in a 3D view in Autodesk Revit.

The tool helps users quickly isolate selected elements in 3D, making it easier to review, coordinate, inspect, and present specific parts of the model.

Designed for everyday BIM workflows, 3D Section Box reduces manual view setup, improves navigation, and speeds up model review in complex Revit projects.

How to Use 3D Section Box?

  1. Select one or more elements.
  2. Click the 3D Section Box button in the BIM Plugin ribbon.
  3. Select an existing 3D view from the list or create a new 3D view.
  4. Choose the 3D Section Box creation mode and enter the required value.

Note: The section box is created based on the selected elements and the values specified in the 3D Section Box dialog.
