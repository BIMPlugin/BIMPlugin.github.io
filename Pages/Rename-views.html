What is Rename Views?

Rename Views is a productivity tool that makes it easy to rename Revit views using a powerful and flexible interface.

Instead of manually editing view names one by one, the tool allows you to quickly rename multiple views using customizable naming options from a single dialog.

Designed for everyday BIM workflows, Rename Views saves time, reduces repetitive work, and helps maintain consistent view naming throughout your project.

How to Use Rename Views?

  1. Click the Rename Views button in the BIM Plugin ribbon.
  2. The Rename Views dialog will open.
  3. Configure the view names the way you need using the available options.
  4. Apply the changes to automatically rename the selected views.

Note: The Rename Views dialog provides flexible options for organizing view names while helping maintain a consistent naming convention throughout the project.
