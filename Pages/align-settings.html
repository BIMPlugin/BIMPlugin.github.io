What is Align Settings?

Align Settings allows you to configure how the Align tools work before performing an alignment operation in Autodesk Revit.

The selected settings are applied to all Align commands, including Align Left, Align Center, Align Right, Align Top, Align Middle, and Align Bottom, ensuring consistent alignment behavior throughout your project.

Designed for everyday BIM workflows, Align Settings provides greater flexibility, improves productivity, and helps maintain consistent documentation standards.

How to Use Align Settings?

  1. Open Align Settings.
  2. Select the alignment mode you want to use.
  3. Save the selected settings.
  4. Run any Align command (Align Left, Align Center, Align Right, Align Top, Align Middle, or Align Bottom) to apply the saved settings.

Note: The selected settings remain active until they are changed, allowing all Align tools to use the same alignment mode.
