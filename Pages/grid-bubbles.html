What is Grid Bubbles?

Grid Bubbles is a productivity tool that quickly toggles the visibility of grid bubbles within a selected area in Autodesk Revit.

Instead of manually turning individual grid bubbles on or off, simply define a rectangular area and BIM Plugin automatically switches the visibility of every grid bubble inside it.

Designed for everyday BIM workflows, Grid Bubbles saves time, simplifies drawing cleanup, and helps maintain clear and consistent documentation.

How to Use Grid Bubbles?

  1. Click the Grid Bubbles button in the BIM Plugin ribbon.
  2. Pick the first corner of the selection area.
  3. Pick the opposite corner to define the rectangular area.
  4. BIM Plugin will automatically toggle the visibility of all grid bubbles inside the selected area. Visible bubbles will be hidden, and hidden bubbles will be shown.

Note: Only grid bubbles located inside the selected area are affected. Existing visibility is automatically toggled for each bubble.
