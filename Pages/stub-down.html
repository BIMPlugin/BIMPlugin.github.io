What is Stub Down?

Stub Down is a productivity tool that automatically creates a downward vertical offset for conduits, cable trays, and ducts. It provides a fast and accurate way to route MEP elements from a horizontal run to a lower elevation.

Instead of manually creating vertical bends, adjusting fittings, and modifying element geometry, Stub Down generates the required transition in just a few clicks, helping you maintain clean and consistent layouts.

Designed for everyday BIM workflows, the tool reduces repetitive modeling tasks, improves productivity, and produces reliable routing for MEP systems in Autodesk Revit.

How to Use Stub Down?

  1. In a 3D view, switch to a Top view or make sure the end of the selected element where you want to create the stub down is pointing away from you. Alternatively, you can use a plan view, where the viewing direction does not matter.
  2. Select the conduits, cable trays, or ducts you want to work with.
  3. Click the Stub Down button in the BIM Plugin ribbon to automatically create the vertical offset.

Note: Before using Stub Down, make sure there is enough space for the vertical offset to be created. If the stub cannot be generated due to Revit geometry or routing constraints, BIM Plugin will display a warning message.
