What is Fill In Settings?

Fill In Settings allows you to choose which parameters will be copied when using the Fill In tool. The selected parameters are automatically transferred to every element within the same run.

Instead of copying every parameter, Fill In Settings gives you full control over which values should be synchronized, making the workflow more flexible and efficient.

Designed for everyday BIM workflows, the tool helps maintain consistent parameter data, reduce manual editing, and improve the quality of Revit documentation.

How to Use Fill In Settings?

  1. Open Fill In Settings and select the parameters you want the Fill In tool to copy.
  2. Save your settings.
  3. Use the Fill In tool to apply the selected parameters to all elements in the same run.
