What is Kick?

Kick is a productivity tool that automatically creates kick offsets for conduits, cable trays, and ducts in Autodesk Revit. It simplifies the process of routing MEP elements around obstacles or changing elevation while maintaining accurate geometry.

Instead of manually creating multiple fittings and adjusting element positions, Kick generates the required offset with just a few clicks, helping you produce clean and consistent layouts.

Designed for everyday BIM workflows, the tool reduces repetitive modeling tasks, improves productivity, and ensures reliable, professional routing for MEP systems.

How to Use Kick?

  1. Select the conduits where you want to create a 90-degree kick.
  2. Click the Kick button in the BIM Plugin ribbon.
  3. In the Kick dialog, select the desired angle to rotate the 90-degree bend and create the kick.
  4. Confirm your selection to generate the kick.

Note: Before using Kick, make sure the selected elements can be physically created in the chosen direction. If the kick cannot be generated due to Revit geometry or routing constraints, BIM Plugin will display a warning message.
