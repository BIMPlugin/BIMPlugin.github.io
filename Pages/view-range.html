What is View Range?

View Range is a productivity tool that allows you to edit the Revit View Range with a real-time preview before applying the changes.

Instead of repeatedly opening the native View Range dialog and checking the results, the tool updates the view instantly as you modify the settings, making the adjustment process much faster and more intuitive.

Designed for everyday BIM workflows, View Range improves productivity, simplifies view configuration, and helps you achieve the desired view settings more efficiently.

How to Use View Range?

  1. Click the View Range button in the BIM Plugin ribbon.
  2. Adjust the View Range settings in the dialog while observing the results in real time.
  3. Continue modifying the settings until you achieve the desired result.
  4. Click OK to apply the changes or Cancel to restore the View Range settings that were active before opening the dialog.

Note: The real-time preview lets you evaluate every change before it is applied. Clicking Cancel restores the original View Range settings.
