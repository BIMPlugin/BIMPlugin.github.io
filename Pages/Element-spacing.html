What is Element Spacing?

Element Spacing is a productivity tool that automatically aligns conduits, cable trays, ducts, and other supported elements using a specified spacing value.

The tool supports two spacing modes: Distance between elements and Distance between centerlines, allowing you to choose the most appropriate alignment method for your project requirements.

Designed for everyday BIM workflows, Element Spacing helps create clean, consistent layouts, reduces repetitive modeling tasks, and significantly improves productivity in Autodesk Revit.

How to Use Element Spacing?

  1. Open Element Spacing Settings and choose the spacing mode: Center-to-Center or Clear Distance (distance between the nearest edges of adjacent elements).
  2. Enter the desired spacing value in inches.
  3. Click the Element Spacing Settings button.
  4. Select the elements you want to align.
  5. Finally, select the reference element that the remaining elements will be aligned to.

Note: The selected reference element remains in its original position, while all other selected elements are repositioned according to the specified Spacing Settings.
