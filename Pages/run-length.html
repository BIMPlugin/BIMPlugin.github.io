What is Run Length?

Run Length is a productivity tool that automatically calculates the total length of a conduit run and the length of each individual segment in Autodesk Revit.

The calculated values can be written directly to user-selected parameters, making it easy to schedule, tag, and document conduit runs without manual calculations.

Designed for everyday BIM workflows, Run Length helps improve parameter consistency, reduces repetitive work, and simplifies project documentation.

How to Use Run Length?

  1. Select one or more conduits.
  2. Click the Run Length button in the BIM Plugin ribbon.
  3. Select the parameter that will store the Total Length Parameter and/or the Segment Length Parameter.
  4. Click Fill Parameter to write the calculated values to the selected parameter(s).

Note: Run Length works with conduit runs. The calculated values are written only to the parameters selected in the Run Length dialog.
