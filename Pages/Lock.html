Lock Lock

What is Lock?

Lock is a convenience tool that keeps the BIM Plugin ribbon panel active while you work in Autodesk Revit.

Normally, Revit automatically switches ribbon tabs depending on the selected command or context. Lock prevents this behavior, allowing the BIM Plugin panel to remain visible and easily accessible during your workflow.

Designed for everyday BIM workflows, Lock reduces unnecessary ribbon switching, improves productivity, and provides faster access to BIM Plugin tools.

How to Use Lock?

  1. Click the Lock button in the BIM Plugin ribbon.
  2. The BIM Plugin ribbon panel will remain active while you work in Revit.
  3. When you want to return to the default Revit ribbon behavior, simply click the Lock button again to disable it.

Note: Lock only affects the behavior of the BIM Plugin ribbon panel. It does not modify your Revit project or any model data.
